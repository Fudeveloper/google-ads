# 品牌词、商标与竞品投放合规手册

更新时间：2026-06-08

本文解释 Ads 套利中 brand bidding、competitor bidding、商标、竞品比较页、affiliate 条款和 Google Ads misrepresentation 之间的关系。目标是让团队在买品牌词、竞品词或写“替代/对比/评测”页面前，能区分平台政策、合同条款、法律/商标风险和用户误导风险。本文不提供绕过 brand bidding 限制、伪装官方、抢注近似域名、仿冒品牌、规避商标投诉、cloaking 或换号继续投放的方案。

## 1. 概念分层

| 概念 | 含义 | 主要风险 |
| --- | --- | --- |
| Brand bidding | 购买广告主、商家、产品或服务的品牌词 | Offer 拒付、投诉、商标限制 |
| Competitor bidding | 购买竞品品牌词或竞品比较词 | 投诉、低相关性、误导风险 |
| Trademark use in ad | 在广告文案、资产或 display URL 中使用商标 | Google 商标投诉和限制 |
| Trademark as keyword | 把商标作为关键词触发广告 | Google 商标政策和合同条款要分开看 |
| Reseller / informational use | 转售、兼容产品、信息型页面中提到商标 | 需要清楚说明身份和页面目的 |
| Impersonation | 暗示自己是官方、授权、合作方或品牌本身 | Misrepresentation / UBP 高风险 |

最重要的原则：**Google Ads 商标政策允许或不限制的事情，不等于 affiliate network、广告主合同或当地法律也允许。** 套利团队必须同时看四层规则。

## 2. 四层规则模型

1. 平台政策：Google Ads trademarks、misrepresentation、counterfeit goods、destination requirements。
2. Offer / affiliate 条款：是否允许 brand bidding、competitor bidding、direct linking、domain bidding、coupon/deal/search/native。
3. 法律和商标权利：不同国家、行业、商标状态和实际使用方式可能不同。
4. 用户体验和证据：用户是否清楚知道你是谁、是否官方、是否授权、推荐是否有商业关系。

只有四层都能解释，才进入测试。任何一层不清楚，都先按高风险处理。

## 3. Google Ads 商标政策要点

Google Ads trademarks policy 说明，Google 会遵守当地商标法律并保护商标所有者权利；当商标所有者提交投诉时，Google 会审核并可能限制广告中商标的使用。

对套利团队最关键的区别：

- Google Ads 商标审核关注“商标是否出现在广告中”，不只看落地页。
- Google Ads 不会因商标作为关键词本身而限制。
- Google Ads 也不会因商标出现在广告 display URL 的二级域名而限制。
- Google Ads 会限制直接竞争者在广告中使用商标。
- Google Ads 会限制以混淆、欺骗或误导方式使用商标。
- 转售、兼容产品、信息型页面或搜索结果索引场景下，商标使用可能被允许，但广告和落地页必须清楚说明是 reseller、informational site 或其他真实身份。

这意味着：买竞品词本身和在广告文案中写竞品商标，是两个不同问题；Google policy、Offer contract 和法律风险也不是同一件事。

## 4. Offer 条款比平台政策更细

Affiliate / CPA / CPL Offer 常见限制：

| 条款 | 解释 |
| --- | --- |
| No brand bidding | 不买广告主品牌词、拼写变体、商标词 |
| No brand + generic | 不买“brand + coupon / review / login / support”等组合词 |
| No competitor bidding | 不买竞品品牌词或竞品比较词 |
| No domain bidding | 不买近似域名、错拼域名、品牌域名 |
| No direct linking | 不直接把广告跳到广告主页面 |
| Pre-approval required | 品牌词、页面、文案必须先审批 |
| Negative brand list | 必须把某些品牌词加入否定词 |

违反 Offer 条款常见结果：

- 转化拒付。
- 月末 scrub。
- 网络账户关闭。
- 广告主投诉。
- 已付款收入被追回。
- 后续同垂类 Offer 难以拿到权限。

因此，即使 Google Ads 允许某种关键词投放，如果 Offer 明确禁止 brand bidding，套利团队也不能做。

## 5. 竞品投放的合规边界

竞品投放可以有正常用途，例如：

- “替代方案”页面：解释功能、价格、适用人群和迁移成本。
- “对比页”：用真实维度比较两个产品。
- “兼容产品”页面：销售配件、替换件或兼容服务。
- “信息页”：解释不同品牌/服务的区别。

但必须满足：

- 页面有原创比较内容，不只是把用户导向 Offer。
- 不暗示自己是竞品官方。
- 不使用竞品 logo、颜色、布局或域名让用户混淆。
- 广告文案不写虚假授权、官方、认证、合作关系。
- 评分、排名、推荐和 affiliate 关系有披露。
- 价格、功能、限制和更新时间可核查。

高风险写法：

- “Official [Brand] Support” 但并非官方。
- “Brand Login / Customer Service” 实际导向第三方 lead 表单。
- “Brand Coupon” 但页面没有真实 coupon。
- “Brand Alternative” 页面没有真实比较，只是桥页。
- 用错拼品牌词、近似域名或仿冒视觉误导用户。

## 6. Misrepresentation 和冒充风险

Google Ads misrepresentation 和 unacceptable business practices 关注广告和目的地是否误导用户。对品牌词投放尤其危险的行为包括：

- 暗示自己得到某品牌、组织、政府或公众人物支持，但没有证据。
- 冒充其他品牌或业务，让用户提交钱或个人信息。
- 没有资质却声称能提供某服务。
- 隐藏业务主体、价格、服务、限制或商业关系。
- 用另一个品牌的名称、logo、图片、颜色或近似拼写让用户误认。

这类问题可能比普通商标限制严重，因为 unacceptable business practices 属于严重政策风险，可能导致账号暂停。

## 7. Counterfeit 和仿牌边界

如果页面销售实体商品、软件、配件或品牌相关服务，还要检查 counterfeit goods policy。高风险信号：

- 商品使用与真实商标相同或极近似标识。
- 页面暗示正品、授权或官方，但没有证明。
- 价格异常低且来源不清。
- 使用品牌图片、包装、认证图标但无法证明权利。
- 用户无法确认卖家身份、退货、保修和授权关系。

套利团队不应碰仿牌、盗版软件、破解授权、假证书或灰色维修服务。短期转化高，长期风险极高。

## 8. 关键词准入和否定词

上线前把关键词分四类：

| 类别 | 示例 | 默认动作 |
| --- | --- | --- |
| 自有品牌词 | 自有品牌、产品名 | 可投，但要保护品牌一致性 |
| 授权品牌词 | 已获书面授权的品牌词 | 按授权范围投放并留证据 |
| 竞品比较词 | `brand alternative`、`brand vs product` | 只在有真实比较页和条款允许时小测 |
| 禁止品牌词 | Offer 禁止、敏感官方、支持/login/coupon 等 | 加否定词 |

否定词清单建议：

- 广告主品牌及拼写变体。
- 竞品品牌及拼写变体。
- `official`、`login`、`support`、`customer service`、`phone number`。
- `coupon`、`promo code`，除非有真实 coupon 授权。
- `scam`、`complaint` 等不适合当前页面任务的词。
- 政府、银行、医疗、技术支持等敏感主体词。

## 9. 页面和广告文案清单

广告文案检查：

- 是否使用了别人的商标。
- 是否暗示官方、授权、合作、认证或背书。
- 是否用竞品名吸引点击但页面没有真实比较。
- 是否误导价格、优惠、可用性或服务范围。
- 是否使用“official”“authorized”“certified”“partner”但无证据。

页面检查：

- About、Contact、Privacy、Terms 是否清楚。
- 是否披露 affiliate/广告关系。
- 是否说明自己不是官方或授权方，除非有授权证据。
- 比较维度、评分方法、价格来源和更新时间是否可见。
- 是否避免复制竞品页面、logo、颜色和布局。
- CTA 是否明确告诉用户会跳到谁、谁会联系或谁收集信息。

## 10. 证据包

每个品牌词或竞品词测试都应保存：

- Offer terms 截图。
- 广告主或 network 邮件确认。
- 品牌词允许/禁止清单。
- 页面审批记录。
- 广告文案审批记录。
- 授权/合作证明链接或合同。
- 否定词清单。
- Search terms report 截图。
- 拒付/投诉/商标限制记录。

没有证据包，不进入放量。

## 11. 决策规则

| 情况 | 动作 |
| --- | --- |
| Offer 明确禁止 brand bidding | 不投，加入否定词 |
| Offer 未说明 brand bidding | 先邮件确认，不默认允许 |
| Google Ads 商标限制广告文案 | 改文案或提交证明，不换号绕过 |
| 广告主投诉竞品词 | 暂停，复核条款和页面 |
| 页面需要写“official/authorized” | 必须有公开可验证授权 |
| 竞品比较页无原创内容 | 不投 |
| 使用品牌词带来高 ROI 但被拒付 | 按 paid/approved 口径重算，停止放量 |

## 12. 系统落地

当前系统支持：

| 业务动作 | 系统位置 |
| --- | --- |
| 记录 Offer 允许/禁止来源、brand bidding 和政策备注 | `/offers` |
| 用关键词研究筛查 brand/competitor/official/support/login 词 | 关键词意图文档和投放草稿 |
| 在投放草稿生成前人工审核关键词和文案 | `/campaigns` |
| 在风险审计记录商标、误导、冒充官方和拒付风险 | `/risk-audits` |
| 保存 Google trademarks、misrepresentation、FTC disclosure 等来源 | `/sources` |

后续可扩展：

- `brand_term_rules`：brand、variant、allowed、prohibited、negative_level、evidence_url。
- `trademark_review_log`：ad text、landing page、risk level、reviewer、decision。
- Campaign CSV 导出前自动提示品牌词和 official/support/login 风险。
- Offer 评分加入 brand bidding clarity 和 evidence completeness。

不做：

- 不绕过 brand bidding 限制。
- 不伪装官方或授权身份。
- 不生成仿冒页面、近似域名或误导性 display URL。
- 不隐藏商标投诉或通过换号继续投放。
- 不用 cloaking 让审核页和用户页看到不同品牌关系。
